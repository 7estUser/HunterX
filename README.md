## 使用方法
需要在和软件的同一目录位置创建配置文件：hunterx.yaml
文件内容：(yaml文件格式，记得冒号后面打一个空格哦)
```yaml
#hunter_userName
userName: 
#hunter_apiKey
apiKey: 
```
